## Directions API

### Endpoint

**GET** `/v1/directions/<start_lon>,<start_lat>;<end_lon>,<end_lat>`

* Query parameters
    
    * **`type`**  
    Accepted values:
        * `driving`
        * `cycling`
        * `walking`
        * `publictransport`
        * `driving-traffic` (*deprecated*)  
        For backward compatibility, equivalent to `driving`.
        
    * **`language`**: Default `en`
